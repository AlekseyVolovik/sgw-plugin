# SGW Plugin

## Релиз новой версии

- Повысить номер версии в основном файле плагина (Version: 1.1.3)
- Залить все изменения в основную ветку main
```bash
git add .
```
```bash
git commit -m "New commit"
```
```bash
git push origin main
```
- Создать тег c номером версии:

```bash
git tag v1.1.3
```

- Запушить тег:

```bash
git push origin v1.1.3
```
